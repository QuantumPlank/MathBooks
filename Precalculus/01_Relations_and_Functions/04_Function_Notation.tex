\section{Function Notation}

\begin{definition}[Function Notation]
    If we think of the domain of a function as a set of \textbf{inputs} and the range as a set of \textbf{outputs}, we can think of a function $f$ as a \textbf{process} by which each input $x$ is matches with only one output $y$. Since the function is completely determined by the input $x$ and thr process $f$, we symbolize the output with \textbf{function notation}: $f(x)$, read 'f of x'.
    The value of $y$ is completely dependent on the choice of $x$. For this reason, $x$ is often called the \textbf{independent variable}, or \textbf{argument} of $f$, whereas $y$ is often called the \textbf{dependent variable}.
\end{definition}
